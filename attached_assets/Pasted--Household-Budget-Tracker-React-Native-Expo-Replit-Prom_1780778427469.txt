# Household Budget Tracker — React Native (Expo) Replit Prompt

## PROJECT SETUP

Create a **React Native app using Expo (Expo Go compatible)**.

Scaffold with:
```
npx create-expo-app HouseholdBudget --template blank
```

Use **Expo Router** for navigation if available, otherwise use **React Navigation Bottom Tabs**.

### Dependencies to install:
```bash
npx expo install @react-navigation/native @react-navigation/bottom-tabs
npx expo install react-native-screens react-native-safe-area-context
npx expo install @react-native-async-storage/async-storage
npx expo install expo-notifications
npx expo install expo-file-system expo-sharing
```

For Excel export:
```bash
npm install xlsx
```

## CALCULATIONS (`/utils/calculations.js`)

```js
export const getTotalSpent = (expenses, month) =>
  expenses
    .filter(e => e.date.startsWith(month))
    .reduce((sum, e) => sum + e.amount, 0);

export const getRemainingBalance = (budget, totalSpent) => budget - totalSpent;

export const getSavingsProgress = (remainingBalance, savingsGoal) =>
  savingsGoal > 0 ? Math.min((remainingBalance / savingsGoal) * 100, 100) : 0;
```

## EXCEL EXPORT (`/utils/exportExcel.js`)

```js
import * as XLSX from 'xlsx';
import * as FileSystem from 'expo-file-system';
import * as Sharing from 'expo-sharing';

export const exportToExcel = async (expenses, budget, month) => {
  const rows = expenses
    .filter(e => e.date.startsWith(month))
    .map(e => ({
      Date: e.date,
      Category: e.category,
      Description: e.description || '-',
      Amount: `Rs. ${e.amount}`
    }));

  // Add summary rows
  const totalSpent = expenses.reduce((s, e) => s + e.amount, 0);
  rows.push({}, 
    { Date: 'Total Budget', Category: `Rs. ${budget}` },
    { Date: 'Total Spent', Category: `Rs. ${totalSpent}` },
    { Date: 'Remaining', Category: `Rs. ${budget - totalSpent}` }
  );

  const ws = XLSX.utils.json_to_sheet(rows);
  const wb = XLSX.utils.book_new();
  XLSX.utils.book_append_sheet(wb, ws, `${month} Statement`);
  const wbout = XLSX.write(wb, { type: 'base64', bookType: 'xlsx' });

  const uri = FileSystem.documentDirectory + `Statement_${month}.xlsx`;
  await FileSystem.writeAsStringAsync(uri, wbout, { encoding: FileSystem.EncodingType.Base64 });
  await Sharing.shareAsync(uri);
};
```

## CALCULATIONS RULES

Always compute fresh, never cache:

```js
const currentMonth = new Date().toISOString().slice(0, 7); // "2026-06"
const monthExpenses = data.expenses.filter(e => e.date.startsWith(currentMonth));
const totalSpent = monthExpenses.reduce((s, e) => s + e.amount, 0);
const remaining = data.budget - totalSpent;
const savingsProgress = data.savingsGoal > 0 ? (remaining / data.savingsGoal) * 100 : 0;
```

## APK BUILD (Download & Install on Android)

Replit cannot build APKs locally. Use **EAS Build** (Expo's free cloud build service) to generate a downloadable `.apk` file.

### Step-by-step after Replit generates the code:

**Step 1 — Create free Expo account**
Go to: https://expo.dev/signup

**Step 2 — Install EAS CLI**
In Replit Shell:
```bash
npm install -g eas-cli
eas login
```

**Step 3 — Configure EAS**
```bash
eas build:configure
```
This creates `eas.json`. Make sure it contains:
```json
{
  "build": {
    "preview": {
      "android": {
        "buildType": "apk"
      }
    }
  }
}
```

**Step 4 — Build the APK (free, cloud-based)**
```bash
eas build -p android --profile preview
```
- No Android Studio needed
- No local Java/Gradle needed
- Build runs on Expo's servers (~10–15 minutes)
- You get a download link when done

**Step 5 — Download & Install**
- Open the link on your Android phone
- Download the `.apk` file
- Go to Settings → Install Unknown Apps → Allow
- Install and open

### Add this as a script in `package.json`:
```json
"scripts": {
  "start": "expo start --tunnel",
  "build:apk": "eas build -p android --profile preview"
}
```

### Add to `README.md`:
```
## To test on phone (instant):
1. npm run start
2. Scan QR with Expo Go app

## To build APK (installable):
1. npm install -g eas-cli
2. eas login  (create account at expo.dev)
3. npm run build:apk
4. Download APK from the link printed in terminal
5. Install on Android phone
```
